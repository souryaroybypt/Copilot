React Calculator App

This is a simple calculator application built with React. It provides a clean and bold UI for basic arithmetic operations.

Features

  • Addition (+)
  • Subtraction (-)
  • Multiplication (ร)
  • Division (รท)
  • Delete last character (DEL)
  • Clear all input (C)
  • Decimal point (.)
  • Numbers 0-9
  • Equals (=) to calculate the result
  • Results and input are clamped to 6 decimal places

Usage

  1. Click the number buttons to enter values.
  2. Use the operator buttons for arithmetic operations.
  3. Press . to add a decimal point (max 6 decimal places per number).
  4. Press = to get the result (rounded to 6 decimal places).
  5. Use DEL to delete the last character, or C to clear all input.

Tech Stack

  • React
  • CSS for styling

How It Works

The calculator restricts decimal input to 6 places per number and displays results rounded to 6 decimal places. It supports chained operations and error handling for invalid expressions.
